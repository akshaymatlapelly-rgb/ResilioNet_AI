🌉 ResilioNet AI
Smart Bridge Health Monitoring System

ResilioNet AI is an AI and IoT based bridge monitoring system designed to improve public safety through real-time structural health analysis.

🚀 Features

Real-time Temperature Monitoring
Water Level Detection
Vibration Monitoring
Structural Load Analysis
AI Risk Score Calculation
Live Web Dashboard
Early Warning Alerts
